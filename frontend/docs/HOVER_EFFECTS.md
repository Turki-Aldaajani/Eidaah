# Hover Effects

كل قواعد `:hover` في الملف المرجعي `referenceeidaah-mockup/eidaah-injaz.html`،
مجمّعة حسب فئة المكوّن.

## البطاقات (Cards)

| العنصر | عند hover |
|---|---|
| `.card` (عام) | الحدود → `--line2`، الخلفية → `--bg-h` |
| `.grade-card` | حدود خضراء + ظل، والرقم `.g-num` يتلوّن + يظهر `.g-cta` (opacity 0→1) |
| `.subject-card` | الحدود تتلوّن بلون المادة `--c`، ويظهر السهم `.s-arrow` وينزلق للداخل |
| `.chapter-card` / `.lesson-card` | خلفية `--bg-h`، ورقم/مؤشر الفصل أو الدرس يتلوّن بلون المادة |
| `.lesson-card` → `.l-go` | الزر الجانبي "ادرس الآن" ينقلب لخلفية معبأة (`--pri` + نص أبيض) |
| `.vid-card` | حدود أوضح + ظل متوسط، وزر التشغيل `.th-play` يكبر `scale(1.08)` |
| `.path-card` / `.feat-card` | يرتفع للأعلى `translateY(-3px إلى -4px)` + حدود + ظل أقوى |
| `.soon-card` | الحدود المتقطعة تتحول للون الأساسي الفاتح فقط (بدون رفع) |
| `.tm-card` (فريق العمل) | يرتفع `translateY(-2px)` + حدود + ظل |

## الأزرار والروابط

| العنصر | عند hover |
|---|---|
| `.btn` | الخلفية تغمق من `--pri` إلى `--pri-h` |
| `.btn.ghost` | الخلفية الشفافة تمتلئ بـ `--pri-soft` |
| `.home-id-btns .btn` (لوحة الهوية الداكنة) | الأخضر الزيتوني يغمق قليلاً |
| `.home-id-btns .btn.ghost` | خلفية بيضاء شفافة خفيفة `rgba(255,255,255,.12)` |
| `.crumbs a` (فتات الخبز) | اللون ينتقل إلى `--pri` |
| `.faq-q` | لون النص ينتقل إلى `--pri` |
| `.faq-mail` | الخلفية تغمق إلى `#D8E7E1` |
| `.v-watch` (زر مشاهدة الفيديو) | يغمق كالزر الأساسي |
| `.cta-btn` (زر CTA السفلي) | يتحول إلى أخضر أفتح `#22995F` |

## عناصر التحكم والفلاتر

| العنصر | عند hover |
|---|---|
| `.fsel select` | الحدود تتحول إلى `--faint` |
| `.fchip` | الحدود تتحول إلى `--pri` |
| `.freset` (إعادة تعيين الفلاتر) | اللون + خلفية `--pri-soft` |
| `.theme-btn` (تبديل الوضع الليلي) | الحدود واللون ينتقلان إلى `--pri` |
| `.opt` (خيار في الاختبار) | حدود + خلفية `--bg-h` (فقط إن لم يكن معطّلاً) |
| `.upz` (منطقة رفع الملف) | حدود `--pri` + خلفية `--pri-soft` (نفس حالة السحب `.drag`) |

## تفاعلات دقيقة (Micro-interactions)

| العنصر | عند hover |
|---|---|
| `.ai-toggle:hover .ai-tt b` | عنوان أداة الذكاء الاصطناعي يتلوّن بالأساسي |
| `.fbar:hover` (شريط التصفية) | خلفية بطاقة + حدود عادية |
