# Eidaah

Eidaah is an AI-powered assistant that helps presenters, students, and educators
understand and deliver presentation slides more effectively.

## Core Features
- Automatic slide content summarization
- Real-world example generation
- Audience understanding questions

## Project Structure
- backend/   : API & AI logic
- frontend/  : Web interface
- model/     : AI models
- docs/      : Documentation & design notes

## Project Status
Early development (MVP)
